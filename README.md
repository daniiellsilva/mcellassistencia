# 📱 MCELL - Site Institucional

Este projeto foi desenvolvido para um cliente que buscava fortalecer sua presença digital e facilitar o contato com seus clientes através de um site moderno, responsivo e objetivo.

A **MCELL** é uma empresa localizada em Belo Horizonte - MG, especializada em assistência técnica para smartphones, venda de celulares e comercialização de acessórios como capas, películas, carregadores, cabos, fones de ouvido e muito mais.

O principal objetivo do projeto foi criar uma página de apresentação da empresa, destacando seus serviços, produtos e facilitando a solicitação de orçamentos através do WhatsApp.

---

## 🎯 Objetivos do Projeto

- Apresentar a empresa de forma profissional;
- Divulgar os serviços de assistência técnica;
- Exibir os principais acessórios comercializados;
- Facilitar o contato direto via WhatsApp;
- Informar localização e redes sociais;
- Criar uma experiência agradável para dispositivos móveis e computadores.

---

## 🚀 Funcionalidades

- ✅ Banner principal com slider automático
- ✅ Mensagens promocionais dinâmicas
- ✅ Navegação com rolagem suave entre seções
- ✅ Seção institucional "Sobre a MCELL"
- ✅ Catálogo de serviços com imagens ilustrativas
- ✅ Catálogo de produtos
- ✅ Botões de orçamento via WhatsApp
- ✅ Endereço com botão para Google Maps
- ✅ Carrossel de avaliações fictícias dos clientes
- ✅ Layout totalmente responsivo
- ✅ Animações suaves durante a navegação
- ✅ Menu fixo com destaque da seção atual

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome
- Google Fonts (Poppins)

---

## 🎨 Recursos Implementados

Durante o desenvolvimento foram utilizados diversos recursos para proporcionar uma melhor experiência ao usuário:

- Slider automático de banners
- Efeitos de transição entre imagens
- Scroll suave entre seções
- Cards com efeito hover
- Animações utilizando Intersection Observer
- Carrossel infinito de avaliações
- Layout responsivo utilizando Flexbox e CSS Grid
- Botões de ação para WhatsApp
- Integração com Google Maps
